# 🚀 Projeto de Login - Guia do Estudante

Este projeto foi preparado para que você possa focar no desenvolvimento do **HTML** e **CSS** da sua tela de login, enquanto a integração com o banco de dados na nuvem (**Supabase**) já está pré-configurada.

## 📦 Como começar

1. **Clone este repositório** no seu computador usando o terminal ou o GitHub Desktop:
   ```bash
   git clone https://github.com/adriel-prof/Aula-Tela-Login.git
   ```
2. **Abra a pasta** no VS Code.
3. **Configure seu Backend**: Siga o [GUIA_DO_ESTUDANTE_SUPABASE.md](GUIA_DO_ESTUDANTE_SUPABASE.md) para criar seu projeto no Supabase e conectar ao seu código.

## 📂 O que tem em cada arquivo?

- `index.html`: Sua estrutura da página de login. **É aqui que você trabalha!**
- `style.css`: Seu design e cores. **Solte a criatividade aqui!**
- `script.js`: Lógica de login e integração. Você só precisará colar suas chaves aqui.
- `register.html` & `register.js`: **[PRONTOS]** Tela de cadastro já funcional.
- `home.html`: **[PRONTO]** Tela interna que o usuário vê após logar.

## 💡 Dicas importantes

- **Não precisa de servidor local**: Como estamos usando o Supabase via CDN, você pode abrir o `index.html` diretamente no navegador ou usar a extensão "Live Server".
- **Edite e recarregue**: Sempre que mudar seu HTML ou CSS, basta salvar o arquivo e atualizar a página no navegador.
- **Foco no Design**: O objetivo desta aula é que cada estudante crie uma interface única e funcional.
